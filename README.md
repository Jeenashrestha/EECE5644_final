**Predicting 30-Day Hospital Readmission (Diabetes)**

This project builds a machine learning pipeline to predict whether a diabetic patient will be readmitted within 30 days using the UCI Diabetes 130-US Hospitals dataset.

**Project Overview**
Task: Binary classification (30-day readmission)
Model: XGBoost (best performing)
Evaluation: PR-AUC, ROC-AUC, F1, precision, recall
Final Results
PR-AUC: 0.2304
ROC-AUC: 0.6698
F1 Score: 0.2774
Recall: 0.4594

**How to Run (Google Colab)**
1. Open JeenaShresthaEECE_finalproject.ipynb in Google Colab
2. Download the dataset from UCI: https://archive.ics.uci.edu/ml/datasets/diabetes+130-us+hospitals+for+years+1999-2008
3. Upload the dataset files to Colab
4. Run all cells in order
